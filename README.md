Instagram Challenge
===================

## User stories

```
As a person
So I can share stuff that has happened in my life
I want to be able to publish photos on a website

As a person
So I can ensure my photographic genius gets the recognition it deserves
I need to be able to sign in to the website

As a person
So I can remember the good times
I want to be able to view the photos I have posted

As a person
So I can find out what is happening in other people's lives
I want to be able to view their photos

As a person
So I can demonstrate my enthusiasm for particular photos
I want to be able to 'like' them

As a person
So I can communicate my reactions to particular photos
I want to be able to comment on them
```
